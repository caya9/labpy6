# Sub Rutin / Fungsi (Python) 
## Deskripsi
Praktikum ini mempelajari implementasi fungsi dalam bahasa Python, meliputi fungsi biasa, fungsi dengan return value, argumen default, keyword arguments, arbitrary arguments, dan lambda function. Tugas utama adalah membuat program pengelolaan data nilai mahasiswa menggunakan fungsi.

# Tujuan 
- Memahami konsep dan jenis-jenis fungsi di Python.

- Mengimplementasikan fungsi dalam program sederhana.

- Membuat program CRUD data mahasiswa menggunakan fungsi.

# Fitur Program
Program terdiri dari empat fungsi utama:

1. tambah() – Menambahkan data mahasiswa

2. tampilkan() – Menampilkan data mahasiswa

3. hapus(nama) – Menghapus data mahasiswa berdasarkan nama

4. ubah(nama) – Mengubah data mahasiswa berdasarkan nama

# Flowchart
``` python
[Mulai]
    ↓
[Tampilkan Menu]
    ↓
[Input Pilihan]
    ↓
──────────────────────────────
| 1. Tambah → Input Data     |
| 2. Tampil → Tampil Data    |
| 3. Ubah  → Input Nama      |
| 4. Hapus → Input Nama      |
| 5. Keluar                   |
──────────────────────────────
    ↓
[Kembali ke Menu?]
    ↓
[Selesai]
```

# Kesimpulan
Dari praktikum ini dapat disimpulkan bahwa penggunaan fungsi mempermudah penyusunan program, membuat kode lebih terstruktur, modular, dan mudah dikembangkan. Fungsi sangat penting dalam pembuatan program Python, terutama untuk mengelola data dan menghindari pengulangan kode.
